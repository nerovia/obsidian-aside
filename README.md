# Obsidian Aside

## Installation

## Usage
  
````
```aside
thumbnail: '![[latissa.jpg]]'
sortContent: false
content:
 full name: 'Latissa Ojeda'
 gender: Female (she/her)
 alias:
  - The Scarred
  - 'Tiss (Strix)'
 race: 'Human'
 class: 'Warlock (Seabound)'
 location: '[[Brineridge]]'
 affilations:
  - '[[Captains Guild]]'
  - '[[Alamilla Lascaris]]'
  - '[[Cordozar]] (first mate)'
  - '[[Arbuur Rosevolt]]'
 appearances:
  - '[[Sessions#Session 11]]'
  - '[[Sessions#Session 15]]'
  - '[[Sessions#Session 17]]'
```
````

| Property      | Description                                                        |
| ------------- | ------------------------------------------------------------------ |
| `thumbnail`   | A markdown image link                                              |
| `content`     | An object with the properties that are to be displayed.            |
| `sortContent` | Wether the properties of `content` are to be sorted alphabetically |
